Graycot.com

Graycot.com is my first endeavor into hosting a website on Netlify, purchasing a domain name, and connecting the domain name to a website! I intend to test my experimental projects here, as the host is much more flexible than Neocities. For example, In VSCode I can click the sync button to push my commits to a github repository, which will be noticed by Netlify and cause the website to update with the new changes almost immediately. This is extreamly useful, as I do not need to interrupt my workflow to copy the project files and paste them into the website directory each time I wish to publish new changes.

Additionally, this site will be act as quarantine zone. Experimental ideas with the potential for damage will not interfere with Gray's Tea.

Free CSS Grid templates

I have made a few CSS Grid templates to help people position items on their webpages. Additionally, I account for various screen sizes, including large monitors, vertical monitors, normal desktops, tablets, and mobile phones. All templates are completely responsive. The majority of the templates feature dedicated layouts for Desktop, Tablet, and Mobile. I had started to gather a small collection of these example grids that I have created, and then I realized that it would be a great idea to share them here!

Simply click the picture of the template you would like to view (all the pictures will be in Desktop layout) and you will be taken to a codepen where you can change the size of the browser to see how the layouts change between Desktop, Tablet, and Mobile. The HTML & CSS will also be avaliable to copy.
